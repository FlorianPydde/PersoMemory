# PersoMemory Weekly Consolidation

Use the persomemory-agent.

Run a draft weekly consolidation.

Read:

1. `DREAMS.md`
2. Daily notes since the last consolidation point.
3. Related project, people, pattern, decision, toolkit, and career notes only when needed.

Score candidate memory by:

1. Frequency.
2. Durability.
3. Actionability.
4. Relevance.
5. Reuse potential.

Return a draft report with:

1. Durable promotions recommended.
2. Project or commitment closures recommended.
3. Stale or noisy memory to disregard.
4. Reusable patterns.
5. Career evidence candidates.
6. Decisions that need Florian approval.

Do not mutate durable memory, close projects, close commitments, create career evidence, or edit `MEMORY.md` without Florian approval.
